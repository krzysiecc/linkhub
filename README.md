# LinkHub

A minimalist personal link hub — one screen, a few links, an interactive
flashlight-on-dark backdrop. Built as the landing page for
**[links.wilnicki.com](https://links.wilnicki.com)**.

## Overview

A single-page profile that gathers professional links (LinkedIn, Credly,
GitHub, CV) behind a Swiss-style typographic layout, animated text, a Three.js
background, and a press-`L` special effect.

## Languages

The whole UI is available in three languages, switchable in-page:

🇵🇱 **polski** · 🇬🇧 **english** · 🇩🇪 **deutsch**

## Development

```bash
npm install
npm run dev      # local dev server
npm run build    # type-check + production build
npm run preview  # preview the build
```
